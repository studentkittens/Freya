Detailed Description

The Database Browser.

In order to use the Client's fill_filelist function it inherits from AbstractFilebrowser, Since it is a Browser itself it also inherits from AbstractBrowser so it can be added to the Browserlist But instead from filling the data directly from the client DatabaseCache is used (over the Proxy pattern)
