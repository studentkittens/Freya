Detailed Description

Acts as a Proxy betwenn Browser::Database and MPD::Client, but caches requested data.

Once the Database requests a list of files for a certain path, a hashmap is looked up if it's already cached, if not DatabaseCache asks the client for the filelist.

As Hashmap a std::map was used, with the Path as key and a vector of values as Hashvalue.
