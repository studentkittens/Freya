Detailed Description

The Base to MPD::Client.

BaseClient does not offer any special command it can send to the server, but is able to set up everything related to connections, i.e setting up the Listener and the connection or reconnection once connection is lost connect() is not a public method, since you are not supposed to instanciate this class, MPD::Client offers the connect() you want to use.

The splitting between BaseClient and Client was done to allow Client Extensions, i.e. AudioOutput, that implements certain mpd commands to enable audio outputs.
