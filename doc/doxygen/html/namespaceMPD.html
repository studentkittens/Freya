Detailed Description

Contains everything related to Interaction with the MPD Server.

This class exists to allow use of the Proxy Pattern (i.e. a Proxy for MPD::Client)

Instead of talking directly with the client in order to request data some classes want to do implement something own, like for example a caching of the data.

An example would be the DatabaseCache that implements this class. It caches all data, and requests data only if not cached yet.
